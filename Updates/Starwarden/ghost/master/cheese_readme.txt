○まうすをいぢるSAORI-basic

○こまんど一覧

　ほとんど第１引数だけ。
　set系とmove系だけは第２〜第４も。

get
getx
gety

　カーソル位置を取得。

set x y t
setx x t
sety y t

　カーソル位置を指定。
　xが横座標、yが縦座標、tが移動時間（単位はミリ秒）。
　tは省略可能。

move x y t
movex x t
movey y t

　カーソル位置を相対設定。
　引数はset系と同じ。

lclick
ldclick
ldown
lup
rclick
rdclick
rdown
rup
mclick
mdclick
mdown
mup

　ボタンエミュレート。
　lは左、rは右、mは中。
　clickはdownしてup、dはそれを２回。

wheeldown
wheelup

　ホイールエミュレート。
